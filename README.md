# BioDex - The Pocket Zoologist AI

BioDex is an AI-powered zoology assistant built with Streamlit that can identify animals from images, fetch biological research, display habitat maps, and provide organism-related insights.

## Short Description

## Features

- Animal and organism image classification using Vision Transformer (ViT)
- Secure login authentication using Streamlit Authenticator
- PubMed-based genetic and biological research retrieval
- DuckDuckGo search integration for general facts
- GBIF habitat and geographical distribution mapping
- Interactive maps using Folium
- Tailwind CSS styling inside Streamlit
- Local model loading support for faster inference
- Confidence filtering for valid organisms and species
- Automatic image resizing to reduce workload and improve speed

## Note

BioDex is currently a prototype project built for experimentation, learning, and demonstrating how multiple AI, biological research, and mapping tools can be combined into a single application.

Some features may still be incomplete, experimental, or dependent on local files such as model weights, authentication configs, and API keys.

This project is mainly intended to showcase:

- AI-powered image classification
- Biological research retrieval
- Organism habitat visualization
- Streamlit application design
- Integration of multiple APIs and tools into one workflow

Future versions may include:

- Better species prediction accuracy
- More advanced filtering for non-animal objects
- Improved UI and dashboards
- More detailed biological reports
- Multi-image support
- Exportable PDF reports
- More powerful zoology-specific knowledge systems
